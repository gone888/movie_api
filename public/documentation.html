Documentation

This will be the server-side of an application that will allow users to access information about different movies, direcors, or genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

API Endpoints

Get a list of all movies /movies GET None None A JSON object holding a list of all movies in the database.
Get a list of all users /users GET None None A JSON object holding a list of all users in the database.

Create a new user /users POST None A JSON object with the data of the user to be added. Example:
{
"Username": "Bryan",
"Password": "bryantestpass",
"Email": "bryan123123@gmail.com",
"Birthday": "2001-11-20"
}
 A JSON object holding the users info that was added including id. Example:

"__v": 0

Add a movie to a users FavoriteMovies array /users/{username}/movies/{movieID} POST username, movieID None A message indicating whether the specified movie had been succesfully added to the users FavoriteMovies array.
